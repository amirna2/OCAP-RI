Description: 
This test resizes and displays in each corner the HD image.  The test
needs to be configured to tune to a High Definition source.

Configuration Parameters:
#High definition Channel
HDFrequency=639000000
HDProgramNum=4
HDQAM=16

Test Control:
1-Scale Upper Left, places the image in the upper left qaudrant.
2-Scale Upper Right, places the image in the upper right qaudrant.
3-Scale Lower Left, places the image in the lower lef quadrant.
4-Scale Lower Right, places the image in the lower right quadrant.
5-Full Screen Video, places the image in full screen mode.
6-Toggle Lang Pref - Fav, changes audio streams.


Evaluate by viewing the tv screen. The image should shift and resize
as indicated above.  The language option should change.


Xlet Control:
initXlet
	read parameters from configuration file
startXlet
	process controls keystrokes
pauseXlet
	clear video
distroyXlet
	clear video
