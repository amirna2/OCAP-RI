Description: 
Blue banner rectangle the upper section of the screen, with the menu text: 
"Press PLAY to start animation, Press SELECT to toggle video". 
The test animation time calculation, denoted as 'Time" is displayed below the menu text. 
This value represents Total time of test, Number of frames, and animation frame rate. 
The default values are '0'. 

Configuration Parameters:
graphic_sourceid used to tune video for w/video timing

Test Control: 
1. Press play button on remote control: 
2. Animation of person running across the screen displays. 
3. Total time, total frames, and frame rate will be updated below the menu screen. 
Example: 
Time took: 640 milliseconds Total frames shown: 42, Frame rate: 65.625) 
4. Perform between 10 and 20 iterations of the tests and record the values. 
5. Calculate the mean and standard deviation for the frame rate value. 
6. Record values for future use.

Test Control:
-none-

Evaluate by viewing the tv screen and recording
time and frame rate.

Xlet Control:
initXlet
	read configuration parameters from configuration file
startXlet
	display test control message graphic
pauseXlet
	none
distroyXlet
	none
