This package provides the default implementation(s) of the {@link org.cablelabs.impl.manager.SignallingManager}. The {@link org.cablelabs.impl.manager.signalling.SignallingMgr} class serves as an abstract base class for all potential implementations provided by this package.

Possible implementations include:

  1. Based upon DAVIC Section Filtering APIs
  2. Based upon native MPE APIs
  3. Based upon native platform-specific APIs
  4. A combination of the above

The prototypical implementation is that written to the available Java APIs, in particular the DAVIC Section Filtering APIs. @since Gear 1.0 @author Aaron Kamienski @version $Revision$, $Date$
