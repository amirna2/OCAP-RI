The org.cablelabs.impl.manager.graphics package represents the main implementation of the {@link org.cablelabs.impl.manager.GraphicsManager} manager interface.

GfxMgr

The {@link org.cablelabs.impl.manager.graphics.GfxMgr} class implements the GraphicsManager interface. It causes initialization of both the AWT and HAVi subsystems.

The ThreadGroup within which the AWT event dispatch thread(s) get created is determined by the Thread that causes AWT to be initialized. As such, it is recommended that the GraphicsManager is auto-started at ManagerManager startup. This will ensure that the ThreadGroup for the AWT event thread(s) is the system thread group (rather than being the application thread group for the first application to reference AWT/HAVi). @since OCAP 0.6 @author Aaron Kamienski @version $Revision$, $Date$
