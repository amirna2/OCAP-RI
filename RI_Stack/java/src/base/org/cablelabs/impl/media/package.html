This package comprises the implementation of JMF. It contains the following sub-packages:
  • content
    • — implementation of {@link javax.media.Player} classes that are constructed via the {@link javax.media.Manager}.
  • mpe
    • — {@link org.cablelabs.impl.manager.Manager} classes that provide native interface to the MPE Media and DVR APIs.
  • player
    • — player framework and implementation classes.
  • presentation
    • — presentation layer classes used by players.
  • protocol
    • — implementation of {@link java.media.protocol.DataSource} class that are constructed via the {@link java.media.Manager}.
  • session
    • — session layer classes used by presentation classes.
  • source
    • — {@link javax.media.protocol.DataSource} framework and implementation classes.
  • vbi
    • — classes that provide VBI functionality.
