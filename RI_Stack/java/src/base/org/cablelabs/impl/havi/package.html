The org.cablelabs.impl.havi package (and its sub-packages) contains additional classes that support the CableLabs HAVi implementation. This includes common support classes as well as CableLabs port-specific implementations.

Package Specification

The architecture of the CableLabs HAVi implementation is separated into two distinct portions. These are the core implementation and the support implementation corresponding to the {@link org.havi.ui} and org.cablelabs.impl.havi package hierarchies. The org.cablelabs.impl.havi package hierarchy is divided into three main subdivisions:
  • implementation support classes
  • porting layer classes
  • port implementation classes

Implementation Support Classes

The implementation support classes are those classes that provide core implementation services to core org.havi.ui classes or default/standard implementation support to various CableLabs HAVi ports.

The implementation support classes found in this package are:

  • {@link org.cablelabs.impl.havi.AwtBackgroundImage}

  • {@link org.cablelabs.impl.havi.DefaultVirtualKeyboard}
  • {@link org.cablelabs.impl.havi.DBFrame}
  • {@link org.cablelabs.impl.havi.DBPanel}
  • {@link org.cablelabs.impl.havi.DVBContainer}
  • {@link org.cablelabs.impl.havi.ImageRender}
  • {@link org.cablelabs.impl.havi.JMFSound}

Porting Layer Classes

The porting layer classes are the portion of the CableLabs HAVi architecture which make CableLabs's HAVi implementation easily portable. It contains interface and base class definitions which greatly simplify the task of porting HAVi to a new system. The central figure in the porting layer is the HaviToolkit abstract base class. By specifying a particular implementation of HaviToolkit one specifies the CableLabs HAVi port implementation to be used.

The classes that make up the porting layer are:

  • {@link org.cablelabs.impl.havi.CapabilitiesSupport}
  • {@link org.cablelabs.impl.havi.DefaultScene}
  • {@link org.cablelabs.impl.havi.DefaultSceneFactory}
  • {@link org.cablelabs.impl.havi.DisplayMediator}
  • {@link org.cablelabs.impl.havi.ExtendedBackgroundConfiguration}
  • {@link org.cablelabs.impl.havi.ExtendedBackgroundDevice}
  • {@link org.cablelabs.impl.havi.ExtendedGraphicsConfiguration}
  • {@link org.cablelabs.impl.havi.ExtendedGraphicsDevice}
  • {@link org.cablelabs.impl.havi.ExtendedScreen}
  • {@link org.cablelabs.impl.havi.ExtendedScreenDevice}
  • {@link org.cablelabs.impl.havi.ExtendedStillImageBackgroundConfiguration}
  • {@link org.cablelabs.impl.havi.ExtendedVideoConfiguration}
  • {@link org.cablelabs.impl.havi.ExtendedVideoDevice}
  • {@link org.cablelabs.impl.havi.HaviToolkit}

Port Implementation Packages

The generic Java Platform port can be found in the org.cablelabs.impl.havi.port.generic package. The port of HAVi to the MPE-based OCAP platform can be found in the {@link org.cablelabs.impl.havi.port.mpe} package.
