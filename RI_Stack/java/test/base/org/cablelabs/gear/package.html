The org.cablelabs.gear package (and its sub-packages) contains the classes that make up the SNAP2 GEAR product. The package contains the SNAP2 GEAR classes that are generally target independent; they depend on the core Java language and libraries only. Sub-packages are used to further divide the SNAP2 GEAR product.

Package Specification

The SNAP2 GEAR package contains the following sub-packages:
  • GEAR Data ({@link org.cablelabs.gear.data})
  • GEAR Utilities ({@link org.cablelabs.gear.util})
  • GEAR for HAVi ({@link org.cablelabs.gear.havi})
